性格（スキル）

4つの基本的な性格

  • ・クール　：10%の確率でダメージ半減
  • ・元気　　：5%の確率で追加攻撃
  • ・まじめ　：5%の確率でHPを1回復
  • ・やんちゃ：2%の確率でダメージ3倍の強打

他にもこんな性格があるよ。君はどの性格を選ぶ？

  • ・いちかバチか … 1%でダメージ4倍（攻撃時）
  • ・かしこ … 20%で受けるダメージ20%減（被弾時）
  • ・せっかち … 15%で与ダメ+2（攻撃時）
  • ・がんこ … 3%で攻撃を完全無効化（被弾時）
  • ・あつい … 3%で防御無視（攻撃時）
  • ・しぶとい … 20%でHP1で踏みとどまる（致死ダメ時）
  • ・のんびり … 常時 防御+2
  • ・かんよう … 毎ターン3%でHP+1〜3（ターン開始時）
  • ・めがみ … 2%でHP+2＆与ダメ+1（そのターン）
  • ・暗黒おーら … 5%で「攻撃+7 / 素早さ+7 / 自傷3」（ターン開始時ランダム）

…まだまだあるよ！アップデートで順次追加/調整予定。
